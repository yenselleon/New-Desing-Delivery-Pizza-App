# Technology Stack - Pizza Delivery Dashboard

## Frontend
- **Framework:** React 18.3.1 (Vite as build tool)
- **Routing:** React Router DOM
- **UI Components:** Radix UI (Primitives for accessible components)
- **Styling:** Tailwind CSS (for utility-first styling)
- **Form Management:** React Hook Form
- **Icons:** Lucide React
- **Data Visualization:** Recharts
- **Animation/UX:** Embla Carousel (for interactive sliders)
- **State Management:** (Inferred) React Hooks / Context API

## Development Tools
- **Build Tool:** Vite 6.3.5
- **Plugin:** @vitejs/plugin-react-swc
- **Package Manager:** (Detected) npm/pnpm
- **Environment:** Node.js (v20+ recommended)
